  <li>baseline(gensim) : gensim 라이브러리를 사용한 word2vec(skip-gram) & knn</li>

  <li>knn : sklearn 라이브러리를 사용한 knn</li>
  <li>result : 손실 함수에 따른 knn 결과</li>
  <li>word2vec : 손실 함수에 따른 word2vec(skip-gram) 학습</li>
